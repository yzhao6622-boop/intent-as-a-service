# 运行部署脚本的方法

由于 Windows PowerShell 的执行策略限制，有几种方法可以运行部署脚本：

## 方法1：使用批处理文件（最简单）✅

直接双击运行 `deploy.bat` 文件（以管理员身份）。

## 方法2：临时允许执行脚本

在 PowerShell 中运行：

```powershell
# 以管理员身份打开 PowerShell，然后运行：
powershell.exe -ExecutionPolicy Bypass -File .\deploy-windows.ps1
```

## 方法3：修改执行策略（一次性设置）

以管理员身份运行 PowerShell：

```powershell
# 查看当前策略
Get-ExecutionPolicy

# 临时允许（仅当前会话）
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process

# 然后运行脚本
.\deploy-windows.ps1
```

## 方法4：永久修改执行策略（不推荐）

```powershell
# 以管理员身份运行
Set-ExecutionPolicy RemoteSigned
```

然后就可以直接运行 `.ps1` 文件了。

---

## 推荐使用方法1

直接双击 `deploy.bat` 文件即可，它会自动处理执行策略问题。
